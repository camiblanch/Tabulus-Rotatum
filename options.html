Tabulus Rotatum

by Cami Blanch Twitter: @caminblanch

Automatically start Tabulus Rotatum on browser startup.

Amount of time in seconds to wait before starting rotation again after user interacts with webpage. Set to 0 to disable pausing on interaction.

Automatically load URLs from a text file hosted in a URL on the web.
The URL to load the list of URLs from. Must include the http or https prefix.

The list must be formatted with the time in seconds follwed by a semicolon (;) followed directly by the URL. Each new time and URL must be on a new line.
Example:
15;http://www.google.com
10;http://www.gmail.com

Special thanks for Ben Hedrington's open source code that this extension is based off of.
