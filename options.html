Tabulus Rotatum

by Cami Greenall Twitter: @camigreenall

Automatically start Revolver Tabs on browser startup.

The list should have full URLS, each separated by a new line. Intervals should be follwed by the URL.
IMPORTANT: Always include a new line following your last entry. Also, do not attepmt to run with no URLs.
Example: http://www.google.com 3

Special thanks for Ben Hedrington open source code that this extension is based off of.
